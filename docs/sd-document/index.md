# E100 二次开发指南

本页汇总 E100 平台二次开发能力与文档入口，面向科研、教学与算法验证场景。

## 开发范围

E100 平台支持以下三类扩展开发：

1. `LubanCat` 机载电脑开发：ROS/Docker 运行环境、定位链路与飞控对接。
2. `STM32 MCU` 开发：串口协议解析、GPIO 灯控、定时器与中断扩展。
3. `UART 通信节点` 开发：`uart_tx_node` 的状态打包与串口发送逻辑。

## 合规与安全要求

- 遵守当地无人机法规与飞行限制要求。
- 不修改飞控核心安全相关功能。
- 涉及数据采集与传输时，遵守数据安全与隐私保护规范。
